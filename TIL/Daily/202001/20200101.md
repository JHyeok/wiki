Nuxt.js with Typescript

yarn create nuxt-app jaebook-client

yarn add -D @nuxt/typescript\
yarn add ts-node

echo {} > tsconfig.json

mv nuxt.config.js nuxt.config.ts\
mv test/Logo.spec.js test/Logo.spec.ts

`import Logo from '@/components/Logo.vue'` 에서 오류가 발생 시

`ts-shim.d.ts` 추가
```typescript
declare module '*.vue' {
  import Vue from 'vue'
  export default Vue
}
```

Jest Test

`yarn add --dev jest vue-jest @vue/test-utils ts-jest @types/jest`

`tsconfig.json` 수정
```json
{
  "compilerOptions": {
    "types": ["@types/node", "@nuxt/vue-app", "@types/jest"]
    // ...
  },
  "include": [
    "**/*.ts",
    "**/*.vue"
  ],
  "exclude": [
    "node_modules"
  ]
}

```

`jest.config.js` 수정
```js
module.exports = {
  moduleNameMapper: {
    '^@/(.*)$': '<rootDir>/$1',
    '^~/(.*)$': '<rootDir>/$1'
  },
  transform: {
    '^.+\\.ts?$': 'ts-jest',
    '.*\\.(vue)$': 'vue-jest'
  },
  moduleFileExtensions: ['ts', 'js', 'vue', 'json']
};
```
