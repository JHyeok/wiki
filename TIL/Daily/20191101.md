#### python with VScode
https://code.visualstudio.com/docs/python/tutorial-flask

맨 아래 프로젝트 구조 나눈 부분에서\
CLI로 실행하기 위해서는 hello_app 폴더로 들어가서, \
`$env:FLASK_APP = "webapp"`\
환경변수를 지정해준 후에, \
`python -m flask run`\
로 정상적으로 실행 가능하다.
